# Example ML regression object.

ML object of the example study in
`system.file("ads", package = "martini")` and the regression outcome
[martini_outc_regr](https://bayer-group.github.io/bmdi-mlai-martini/reference/martini_outc_regr.md)
as created by
[`prepare_ml()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/prepare_ml.md).

## Usage

``` r
martini_ml_regr
```

## Format

A `martini_ml` list object. See the documentation of
[`prepare_ml()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/prepare_ml.md)
for further details on the object structure.

## Source

Simulated data
