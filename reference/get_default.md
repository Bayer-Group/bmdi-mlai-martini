# get fct defaults

get fct defaults

## Usage

``` r
get_default(fun, arg = NULL, unlist = TRUE, unname = TRUE)
```

## Arguments

- fun:

  function name

- arg:

  character (vector) of names of arguments to show default values for
  (exact match required)

- unlist, unname:

  logicals

## Value

depending on values unlist and unname either a list or a vector
