# Example ML classification object.

ML object of the example study in
`system.file("ads", package = "martini")` and the classification outcome
[martini_outc_class](https://bayer-group.github.io/bmdi-mlai-martini/reference/martini_outc_class.md)
as created by
[`prepare_ml()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/prepare_ml.md).

## Usage

``` r
martini_ml_class
```

## Format

A `martini_ml` object. See the documentation of
[`prepare_ml()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/prepare_ml.md)
for further details on the object structure.

## Source

Simulated data
