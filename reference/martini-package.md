# martini: BMDI MLAI pipeline - Prepare Module

Prepare module of the BMDI MLAI pipeline. It contains functions to
conveniently transform a set of ADaM data sets (adsl, bds and occds)
into a data object that can be used for machine learning, e.g. using the
BMDI MLAI pipeline.
