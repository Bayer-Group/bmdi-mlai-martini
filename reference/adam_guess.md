# Guess role columns for spec

Guess role columns for spec

## Usage

``` r
adam_guess(role, type = c("bds", "occds"), colnames_data)
```

## Arguments

- role:

  the spec role of interest

- type:

  either `bds` or `occds`. defaults to `bds`

- colnames_data:

  character vector, containing the column names of the ADaM data set to
  be processed

## Value

The guessed column name or `NULL`
