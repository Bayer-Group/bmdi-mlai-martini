# create alias for fct_explicit_na and fct_na_value_to_level based on available forcats version

create alias for fct_explicit_na and fct_na_value_to_level based on
available forcats version

## Usage

``` r
fct_na_to_level(f, level)
```

## Arguments

- f:

  A factor (or character vector)

- level:

  Level to use for missing values: this is what NAs will be changed to

## Value

a factor
