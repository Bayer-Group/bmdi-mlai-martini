# Example specification object.

Specification object of the included example study in
`system.file("martini_example_study/ads", package = "martini")` as
created by
[`adam_spec()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/adam_spec.md).

## Usage

``` r
martini_spec
```

## Format

A list with the entries `adsl`, `adlb`, `admh` and `advs`. See the
documentation of
[`adam_spec()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/adam_spec.md)
for further details on the object structure.

## Source

Simulated data
