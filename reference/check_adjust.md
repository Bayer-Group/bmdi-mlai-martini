# check key value pair inputs for adjust\_\* functions

check key value pair inputs for adjust\_\* functions

## Usage

``` r
check_adjust(spec, entry, modifications)
```

## Arguments

- spec:

  spec object to modify

- entry:

  name of list element to modify in `spec`

- modifications:

  list of key value pairs defining adjustments

## Value

subset of modifications that are valid to apply
