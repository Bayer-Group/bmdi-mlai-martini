# Example feature matrix.

Feature matrix of the included example study in
`system.file("ads", package = "martini")` created by
[`build()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/build.md)
from
[martini_spec](https://bayer-group.github.io/bmdi-mlai-martini/reference/martini_spec.md).

## Usage

``` r
martini_feat
```

## Format

A tibble with one row per subject and the features in the column. See
the documentation of
[`build()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/build.md)
for details on the object structure.

## Source

Simulated data
