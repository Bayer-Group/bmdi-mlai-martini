# Example ML survival object.

ML object of the example study in
`system.file("ads", package = "martini")` and the survival outcome
[martini_outc_surv](https://bayer-group.github.io/bmdi-mlai-martini/reference/martini_outc_surv.md)
as created by
[`prepare_ml()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/prepare_ml.md).

## Usage

``` r
martini_ml_surv
```

## Format

A `martini_ml` list object. See the documentation of
[`prepare_ml()`](https://bayer-group.github.io/bmdi-mlai-martini/reference/prepare_ml.md)
for further details on the object structure.

## Source

Simulated data
